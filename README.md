# Robotics Lab 5: Camera Calibration and Stereovision

This repository contains screenshots and results from **Laboratory 5: Camera Optics, Calibration, and Stereovision** at Poznan University of Technology.

### Contents
- Screenshots of the disparity map and calibration results.
- ROS 2 and OpenCV-based camera calibration process.
- Stereovision setup and real-time parameter tuning.

### How to Use
1. Clone this repository.
2. Open the screenshots to see the results of the calibration and disparity map.
3. For detailed steps, refer to the [laboratory instructions](https://example.com).
