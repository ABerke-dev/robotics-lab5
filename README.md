# Robotics Lab 5: Camera Calibration and Stereovision

This repository contains the results and screenshots from **Laboratory 5: Camera Optics, Calibration, and Stereovision** at **Poznan University of Technology, Institute of Robotics and Machine Intelligence**.

---

## **Laboratory Goals**
- Understand the fundamentals of **camera optics** and **lens distortion**.
- Learn **camera calibration** techniques using **OpenCV** and **ROS 2**.
- Explore **stereovision** principles and generate **disparity maps** for depth estimation.

---

## **Contents**
- Screenshots of:
  - Camera calibration results (chessboard pattern detection).
  - Disparity map visualization.
  - Real-time parameter tuning with `rqt_reconfigure`.
- ROS 2 bag files and calibration YAML files (if applicable).

---

## **Tools & Technologies**
- **OpenCV** (Camera Calibration, Stereo Vision)
- **ROS 2** (Camera Info, Bag Files, `camera_calibration` package)
- **Docker** (Containerized ROS 2 environment)
